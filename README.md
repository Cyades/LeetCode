# 🏆 LeetCode Daily Tracker

A collaborative repository for tracking daily LeetCode progress with friends!

## 📋 How to Join

1. **Clone the `Clone Me` folder** and rename it to your name
2. Inside your folder, organize solutions by difficulty: `Easy/`, `Medium/`, `Hard/`
3. Name your solution files with the problem number (e.g., `1.py`, `42.py`)
4. Commit and push your progress daily!

## 🚀 Getting Started

```bash
# Clone the repo
git clone <repo-url>

# Copy the template folder
cp -r "Clone Me" "YourName"

# Start solving and commit!
git add .
git commit -m "Add solution for problem #XX"
git push
```

## 📌 Rules

1. **Daily commitment** - Try to solve at least 1 problem per day
2. **Update the tracker** - Keep the progress table updated
3. **No peeking** - Don't cheat from others' work if you're not done yet
